# Simple JavaScript Pokédex

This project is a simple Pokédex web application that uses the [PokéAPI](https://pokeapi.co/) to display information about Pokémon. Users can browse through a list of Pokémon and click on individual entries to see more details, including height and an image of the Pokémon.

## Getting Started

To get the project running on your local machine, follow these steps:

1. Clone the repository: `git clone https://github.com/DarkVisor5/simple-js-app.git`
2. Change into the project directory: `cd simple-js-app`
3. Open the `index.html` file in your preferred web browser.

## Project Dependencies

This project has the following dependencies:

- [Bootstrap 4.5.2](https://getbootstrap.com/docs/4.5/getting-started/introduction/) for styling
- [jQuery 3.5.1](https://jquery.com/) for DOM manipulation and event handling
- [Promise Polyfill 8](https://www.npmjs.com/package/promise-polyfill) for browser compatibility
- [Fetch Polyfill 2.0.4](https://github.com/github/fetch) for browser compatibility
- [ESLint](https://eslint.org/) for code linting with the following rules:
  - `eslint:recommended`
  - `plugin:jquery/recommended`

## API Used

This project uses the [PokéAPI](https://pokeapi.co/) to fetch information about Pokémon.
